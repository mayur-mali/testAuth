# Project Management Front End

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
# or
$ npm run serve

# build for production and launch server
$ npm run build
$ npm run start

# Analyse dependencies in the project
$ npm run analyse

# Eslint will auto-fix fixable errors still if you want to manually run it
$ npm run lint:fix

# Run test suite
$ npm run test
```
